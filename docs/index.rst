ElectroKitty documentation
==========================

Welcome to ElectroKitty's documentation. ElectroKitty is a Python library made for simulations of electrochemical reactions.

With ElectroKitty you can:

- Simulate complex reaction networks of electrochemical and chemical reactions
- Plot the results
- Fit your model onto data

For a how to guide see the tutorials found on the library GitHub: https://github.com/RedrumKid/ElectroKitty.

If you would like to get in touch see the :doc:`contact page <contact>`
